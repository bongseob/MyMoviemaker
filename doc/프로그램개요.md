# 프로그램 개요 및 실행 가이드

이 문서는 **Antigravity Movie Maker**의 주요 기능과 사용 방법, 그리고 실행 환경에 대한 정보를 제공합니다.

---

## 1. 프로그램 개요
**Antigravity Movie Maker**는 이미지와 오디오 파일을 결합하여 고품질의 MP4 영상을 생성하는 Electron 기반 데스크톱 애플리케이션입니다. 특히 텍스트 입력을 통해 자동으로 영상 길이에 맞춘 자막(.srt)을 생성하고 영상에 합성(Burn-in)하는 기능을 특화하였습니다.

### 주요 기능
- **미디어 결합**: 여러 장의 이미지(JPG, PNG)와 오디오(MP3)를 하나의 영상으로 제작
- **지능형 자막 생성**: 긴 텍스트를 입력하면 영상 전체 시간에 맞춰 슬라이드별로 자막을 자동 분할 및 배치
- **고성능 엔진**: 내장된 FFmpeg을 사용하여 별도의 코덱 설치 없이도 안정적인 인코딩 지원
- **최신 UI**: 다크 테마 기반의 글래스모피즘(Glassmorphism) 디자인 및 직관적인 드래그 앤 드롭 인터페이스
- **유튜브 연동**: 제작된 영상을 유튜브 계정에 바로 업로드할 수 있는 OAuth2 기반 업로드 지원

## 2. 실행 방법

### 개발 환경 설정 및 실행
프로그램을 개발 모드에서 실행하거나 수정하려면 다음 단계를 따르세요.

1.  **의존성 설치**:
    ```powershell
    npm install
    ```
2.  **개발 모드 실행**:
    ```powershell
    npm run electron:dev
    ```
    *   Vite 개발 서버가 먼저 구동되고, 이후 Electron 창이 나타납니다.

### 실행 파일(Production Build) 생성
배포용 단일 실행 파일(.exe)을 만들려면 다음 명령어를 사용하세요.

```powershell
npm run electron:build
```
*   `release` 폴더 내에 `AntigravityMovieMaker [버전].exe` 파일이 생성됩니다.
*   이 파일은 **Portable(무설치)** 버전으로, 어디서나 바로 실행이 가능합니다.

## 3. 사용 가이드
1.  **이미지 추가**: 중앙 대시보드나 '이미지 선택' 버튼을 통해 영상에 들어갈 이미지들을 업로드합니다.
2.  **오디오 추가**: 배경음악으로 사용할 오디오 파일을 선택합니다. (파일이 없으면 기본 3초 정지 영상으로 생성됩니다.)
3.  **내용 입력**: 타이틀 및 자막으로 사용할 내용을 텍스트 영역에 입력합니다.
4.  **영상 설정**: 화면 비율(16:9 또는 9:16) 및 제목 위치를 설정합니다.
5.  **내보내기**: '비디오 내보내기' 버튼을 클릭하면 `YYYYMMDD.mp4` 형태의 파일명으로 저장을 진행합니다.
6.  **업로드(선택)**: 'YouTube 업로드' 탭에서 생성된 영상을 유튜브에 바로 게시할 수 있습니다.

---

## 4. 문제 해결 (Troubleshooting)
-   **실행 파일 에러**: 실행 파일 실행 중 JavaScript 에러(`spawn EFTYPE`)가 발생한다면, 최신 빌드(`npm run electron:build`)를 통해 해결되었는지 확인해 주세요.
-   **FFmpeg 경로**: 프로그램 내부적으로 `ffmpeg-static` 라이브러리를 사용하며, 패키징 시 `app.asar.unpacked` 경로를 통해 환경에 상관없이 바이너리를 실행하도록 설계되어 있습니다.
